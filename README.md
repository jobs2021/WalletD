# MaxiWallet - Billetera Digital

Aplicación web para gestión de transacciones financieras.

## Descripción

MaxiWallet es una billetera digital que permite a los usuarios:
- Iniciar sesión de forma segura
- Depositar dinero
- Enviar dinero a otros usuarios
- Consultar historial de transacciones
- Gestionar su saldo

## Tecnologías

- HTML5
- CSS3
- JavaScript
- jQuery
- Bootstrap 5

## Estructura

```
Wallet digital/
├── index.html
├── login.html
├── menu.html
├── deposit.html
├── sendmoney.html
├── transactions.html
├── css/
└── js/
```

## Cómo usar

1. Abre el archivo `index.html` en tu navegador
2. Haz clic en "Iniciar Sesión"
3. Completa el formulario de login
4. Accede al panel de control
5. Usa las opciones de depósito, envíos y transacciones

## Características

- Validación de formularios
- Interfaz responsive
- Historial de transacciones
- Gestión de saldo
- Efectos visuales y animaciones
